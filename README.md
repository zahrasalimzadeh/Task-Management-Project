# Task-Management-Project
توضیحات پروژه:
این یک پروژه مدیریت تسک و کارمندان است که با زبان پایتون توسعه یافته است. 
به عنوان مدیر عامل، شما می‌توانید:
- کارمندان جدید را با کد پرسنلی ثبت کنید.
- تسک‌های جدید تعریف کرده و به کارمندان اختصاص دهید.
- وضعیت تسک‌ها را بین لیست‌های (انجام نشده، در حال انجام، انجام شده) جابه‌جا کنید.

Project Description:
This is a Task and Employee Management system developed in Python.
As a CEO, you can:
- Register new employees using their personnel codes.
- Create new tasks and assign them to employees.
- Manage and move task statuses between (Undone, Doing, and Done) lists.
